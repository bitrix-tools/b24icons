Библиотека содержит SVG-иконки для Битрикс24.

::card-group

:::card
---
title: Все иконки в одном месте
---
Библиотека предоставляет полный набор SVG-иконок для Битрикс24 для простого и удобного использования.
:::

:::card
---
title: Умная загрузка иконок
---
Компонент B24Icon оптимизирует загрузку иконок, используя динамический импорт, сохраняя при этом эффект «тряски» дерева.
:::
